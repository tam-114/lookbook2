Squad on Site (S.O.S)

Personal Project

The Squad on Site (SOS) mobile app allows gamers to find other likeminded people to play team-based games with, and was a collaborative project that I've developed more on. In this effort, my role was as the designer.
In developing this project, I created a landing page that showcases the features that will be included in the app. This would include the personality test, news feed, and calendar features as found on the wireframe with Adobe XD.
Design choices for the website were to capitalize on a sleek feel for a new mobile app. The graphics were created with Adobe Photoshop, such as the messaging images, phones in the banner, logo, and color changes to the icons.
There is an optional form for contact to be notified of the release date of the app as it is currently under development being built in React Native. The site is mobile and user-friendly using HTML5, CSS3, JavaScript, Bootstrap framework, and the AOS animation library.
